# 🚀 دليل التشغيل خطوة بخطوة | Setup Guide

## ليش الموقع ما يشتغل لما أفتحه مباشرة في كروم؟

الموقع فيه ملفات **PHP** (تسجيل، دخول، حجز، فيدباك). ملفات PHP **ما تشتغل** إلا بسيرفر.
لذلك لازم تثبّت **XAMPP** وتشغّل الموقع عن طريقه — مو بفتح ملف index.html مباشرة.

---

## الخطوات (بالترتيب)

### 1️⃣ تثبيت XAMPP
- نزّل XAMPP من: https://www.apachefriends.org
- ثبّته بشكل عادي (Next → Next → Finish)

### 2️⃣ نسخ المشروع
- انسخ مجلد `alula-vision2030` كامل
- الصقه داخل: `C:\xampp\htdocs\`
- المسار النهائي يصير: `C:\xampp\htdocs\alula-vision2030\`

### 3️⃣ تشغيل الخوادم
- افتح **XAMPP Control Panel**
- اضغط **Start** عند **Apache**
- اضغط **Start** عند **MySQL**
- لازم يصيرون باللون الأخضر ✅

### 4️⃣ إنشاء قاعدة البيانات
- افتح المتصفح واكتب: `http://localhost/phpmyadmin`
- اضغط تبويب **Import** (استيراد)
- اضغط **Choose File** واختر: `database/alula_db.sql`
- اضغط **Go** (تنفيذ)
- بتشوف رسالة نجاح، وقاعدة البيانات `alula_db` تنشأ تلقائيًا

### 5️⃣ فتح الموقع
- في المتصفح اكتب:
  `http://localhost/alula-vision2030/index.html`
- ⚠️ مهم: افتحه بهذا الرابط، **لا تفتح ملف index.html بالضغط عليه مباشرة**

## حل المشاكل الشائعة

**❌ "Connection failed"**
→ تأكد MySQL شغّال (أخضر) في XAMPP Control Panel

**❌ الصفحة تنزّل ملف PHP بدل ما تفتحه**
→ Apache مو شغّال، أو فتحت الملف مباشرة. استخدم رابط `http://localhost/...`

**❌ Port 80 مشغول (Apache ما يبدأ)**
→ سكّر Skype أو غيّر بورت Apache من إعدادات XAMPP

**❌ النص العربي يطلع رموز ؟؟؟**
→ ملف SQL مضبوط على UTF-8، تأكد ما غيّرت إعدادات الاستيراد
