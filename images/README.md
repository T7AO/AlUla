# Images & Media Guide

This file explains what images and videos to add to each folder.

## 📁 images/logo/
Place your institutional logos here:
- `imsiu-logo.png` — Imam Mohammad Ibn Saud Islamic University logo
- `ccis-logo.png` — College of Computer and Information Sciences logo

> Recommended size: 200x80 px, transparent PNG

## 📁 images/backgrounds/
Background image used across all pages:
- `alula-bg.jpg` — A subtle, faded AlUla landscape

> Recommended size: 1920x1080 px

## 📁 images/places/
Photos for each tourist destination:
- `hegra.jpg`
- `oldtown.jpg`
- `elephantrock.jpg`
- `dadan.jpg`
- `profile-placeholder.jpg` — used for the team slideshow

> Recommended size: 800x600 px

## 📁 images/gallery/
Photos for the gallery page:
- `photo1.jpg` through `photo8.jpg`

> Recommended size: 600x400 px

## 📁 videos/
Short videos for the gallery:
- `alula-tour.mp4`
- `hegra-experience.mp4`

> Format: MP4, max 50MB per video

## Where to find free images?

Sources for high-quality, free-to-use AlUla images:
- https://experiencealula.com/en/media-centre (official media)
- https://www.unsplash.com/s/photos/alula
- https://www.pexels.com/search/saudi%20arabia/

Always check the license before using.
